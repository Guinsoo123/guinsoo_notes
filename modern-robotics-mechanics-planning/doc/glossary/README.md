# 全书核心术语总表（中英对照）

| 术语 | 中文释义 | 主要章节 |
|---|---|---|
| SE(3)（三维特殊欧式群） | 三维刚体位姿群。 | chapter-03, chapter-04 |
| Twist（速度旋量） | 刚体瞬时速度的6维表示。 | chapter-03, chapter-05 |
| Wrench（力旋量） | 力和力矩的6维表示。 | chapter-03, chapter-12 |
| Jacobian（雅可比矩阵） | 关节速度到末端速度的映射矩阵。 | chapter-05, chapter-06 |
| Singularity（奇异性） | 雅可比降秩导致能力退化。 | chapter-05, chapter-07 |
| Inverse Kinematics（逆运动学） | 由目标位姿求关节变量。 | chapter-06 |
| Mass Matrix（质量矩阵） | 描述动力学惯性耦合。 | chapter-08 |
| Time Scaling（时间标定） | 几何路径到时间轨迹的映射。 | chapter-09 |
| RRT（快速随机树） | 随机采样树规划算法。 | chapter-10 |
| PRM（概率路图） | 概率路图规划算法。 | chapter-10 |
| Computed Torque（计算力矩控制） | 基于模型补偿的控制方法。 | chapter-11 |
| Force Closure（力封闭） | 抓取可平衡任意外载。 | chapter-12 |
| Nonholonomic（非完整约束） | 不可积分的速度约束。 | chapter-02, chapter-13 |
