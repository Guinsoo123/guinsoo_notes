# 第5章-03 开链机构静力学

## 核心术语

- `statics`
- `joint torque`
- `wrench mapping`

## 一句话定义

- `statics`：研究平衡状态下力与力矩关系。
- `joint torque`：关节驱动器输出的力矩。
- `wrench mapping`：末端受力映射到关节力矩的关系。

## 关键关系/公式（记忆版）

- V=J(theta)q_dot。
- tau = J^T F_tip（静力学对偶关系）。

## 易错点

- 混淆坐标系表达与物理对象本身。
- 未检查公式适用条件（小角度、无约束、线性化等）。

## 与前后章节的连接

- 静力学是速度映射的对偶形式。
- 该节概念通常在后续算法章节中作为输入模型被重复调用。
