## 一、学机器人时，「李群」要掌握到什么程度

多数教材（含本书对应的术语笔记）对李群是 **工具性** 使用：把 \(SO(3)\)、\(SE(3)\) 当作带结构的矩阵群来运算与求导，而非先做完整公理化。

### 必须熟练（与作业、仿真、实现强相关）

| 主题 | 你要能做什么 |
|------|----------------|
| **矩阵群 \(SO(3)\)、\(SE(3)\)** | 写出元素形式、两变换如何复合、逆变换形式；齐次矩阵分块含义清楚。 |
| **角速度 / 线速度 / 雅可比** | 能在「空间 / 物体」或「世界 / 本体」等约定下写对方程，知道变量在哪个坐标系。 |
| **指数与对数** | \(\exp: \mathfrak{so}(3)\to SO(3)\)（Rodrigues）、\(\exp: \mathfrak{se}(3)\to SE(3)\)（twist 的指数）能算，并能对应到物理量（角速度、螺旋轴等）。 |
| **伴随 Adjoint** | 知道为何出现：不同参考系下 twist / wrench 如何互相变换；动力学、控制、手眼标定里会反复遇到。 |

### 重要但可边用边补

- **李括号、BCH 公式**：精细插值、小摄动、部分论文会用到；不必一开始就啃完全部证明。
- **「李群 = 光滑流形 + 群」**：一句话定义即可；完整流形论可后置。

### 通常可以后推

- 一般李群的分类、根系统、表示论。
- 流形上微分形式、李群上不变 Haar 测度（除非走几何力学理论方向）。

**结论**：掌握 **\(SO(3)\)、\(SE(3)\) 的矩阵模型 + 李代数与 \(\exp/\log\) + 雅可比与伴随**，可覆盖现代机器人刚体运动的大部分内容；其它李群多作为推广阅读。

## 二、线性代数：需要哪些先行知识

### 核心（几乎处处用到）

- 向量空间、子空间、线性无关、基、维数。
- 矩阵乘法、转置、秩、可逆、**分块矩阵**。
- **特征值与特征向量**；实对称矩阵的正交对角化。
- **行列式**：与体积、**定向**、\(SO(n)\) 中 \(\det(R)=1\) 的含义。
- **内积、正交、正交矩阵** \(O(n)\) 与 **特殊正交群** \(SO(n)\)。
- **迹** \(\mathrm{tr}(AB)=\mathrm{tr}(BA)\) 等（伴随表示、二次型与动能中会碰到）。

### 很有用（深度取决于你是否读证明）

- **反对称矩阵** 与 \(\mathbb{R}^3\) 叉积的同构（\(\mathfrak{so}(3)\) 的标准矩阵模型）。
- **矩阵指数** \(e^X\)、对数（谱半径与级数收敛的直觉即可）。
- 双线性型、二次型（度量、刚体动能里的正定矩阵）。
- 若读证明：**若当型** 或至少理解「复特征值对应平面旋转块」与 \(SO(3)\) 的几何图像。

### 可暂缓

- 抽象张量、范畴论、Jordan–Chevalley 全部分解细节——按需补。

## 三、微积分与分析：需要哪些先行知识

### 核心

- **多元微积分**：偏导、梯度、**链式法则**。
- 向量值函数 \(\mathbb{R}^n\to\mathbb{R}^m\) 的导数（**Jacobian**）。
- **常微分方程**初值问题：单参数子群 \(\exp(t\xi)\) 与 \(\frac{\mathrm{d}}{\mathrm{d}t}e^{tX}\) 连着理解最顺。

### 进入「李群 = 流形」叙述时常用

- \(\mathbb{R}^n\) 上开集、连续、可微（拓扑只需最低限度：**开集、连续映射**）。
- **反函数定理 / 隐函数定理** 的叙述级理解（说明 \(\exp\) 在单位元附近局部可逆时常被引用）。
- 光滑映射在一点的**线性化（切映射）**与李代数对应。

### 更深（几何力学、理论控制）

- 流形上切空间、向量场、积分曲线（与 \(\exp\)、动力学挂钩）。

## 四、建议的自修顺序（把「李群」落在连续线上）

1. **线代**：内积、正交阵、行列式、特征值 \(\rightarrow\) 立刻接 \(SO(3)\) 的几何意义。
2. **多元微积分 + 矩阵指数**：\(e^{\omega^\wedge}\)、Rodrigues 公式。
3. **ODE**：\(\dot{R}=R[\omega]\) 或 \(\dot{R}=[\omega]R\) 与 \(\exp\) 的对应。
4. **\(SE(3)\)**、twist、齐次变换 \(\rightarrow\) **雅可比、伴随**。

这样「李群」不是孤立抽象块，而是 **线代 + 多元微积分 + ODE** 的自然延伸。
