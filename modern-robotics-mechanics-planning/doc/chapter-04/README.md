# 第4章 Forward Kinematics（正向运动学）

## 本章核心问题

- 本章回答：正向运动学在机器人建模/规划/控制链路中的作用是什么。
- 重点把术语与可计算关系对应起来，而不是死记公式。

## 本章最重要 5 个术语

- `product of exponentials（指数积公式）`：串联机构位姿由多个关节指数映射相乘得到。
- `space frame（空间坐标系）`：固定在基座上的参考坐标系。
- `body frame（本体坐标系）`：固定在末端执行器上的参考坐标系。
- `M matrix（初始位姿矩阵）`：零位姿下末端相对基座的齐次变换。
- `URDF（统一机器人描述格式）`：机器人结构、关节和惯性参数的XML描述格式。

## 本章易错点

- 只记术语名字，不理解变量与坐标系语义。
- 忽略“适用条件”，把局部结论错误外推。
