## 本章核心问题

- 本章回答：轨迹生成在机器人建模/规划/控制链路中的作用是什么。
- 重点把术语与可计算关系对应起来，而不是死记公式。

## 本章最重要 5 个术语

- `trajectory（轨迹）`：随时间参数化的状态曲线。
- `path（路径）`：仅含几何形状、不含时间信息的曲线。
- `time scaling（时间标定）`：将几何路径映射为可执行时间轨迹。
- `spline（样条曲线）`：分段多项式并保持连接处连续。
- `time-optimal（时间最优）`：本章关键概念。

## 本章易错点

- 只记术语名字，不理解变量与坐标系语义。
- 忽略“适用条件”，把局部结论错误外推。
