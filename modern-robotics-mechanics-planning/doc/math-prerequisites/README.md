# 数学先修指南（刚体运动与现代机器人）

## 本章定位

本章是进入 `chapter-03` 之前的数学桥接章节。  
目标不是完整学习抽象李群课程，而是建立一套能直接用于机器人推导与实现的数学工具箱：

- 忘了定义时，1 分钟可回忆；
- 忘了直觉时，能马上联想到几何意义；
- 写公式时，不混淆坐标系和变量语义。

## 学习方式

本章按“类型”拆分为多个小节，每节都按类似教材的节奏组织：

- 定义（What）
- 直觉（Why）
- 最小公式（How）
- 机器人语境（Where）
- 典型坑点（Pitfalls）
