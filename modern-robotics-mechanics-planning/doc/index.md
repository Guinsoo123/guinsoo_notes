# Modern Robotics 术语学习站

这是《Modern Robotics》中文术语化总结站点入口。

建议阅读顺序：

1. [全书总览](./overview.md)
2. [数学先修指南（刚体运动）](./math-prerequisites/README.md)（读第 3 章前建议通读）
3. [数学先修速查（考前 5 分钟）](./math-prerequisites-cheatsheet.md)（考试/推导前快速回看）
4. [全书术语总表](./glossary/README.md)
5. 按章节阅读 `chapter-01` 到 `chapter-13`（章节目录中「数学先修指南」与「速查页」均排在前）

如需本地渲染：

```bash
pip install -r requirements-docs.txt
./scripts/serve_docs.sh
```
